# IAM

App móvil de conexión social para personas neurodivergentes. Facilita encuentros reales entre personas que se entienden.

**Plataformas**: iOS y Android
**Mercado inicial**: Chile
**Stack**: Flutter · NestJS · Supabase · Google Cloud Run

---

## Estructura del Proyecto

```
apps/
  mobile/     # Flutter app (usuarios)
  backend/    # NestJS API
packages/
  shared/     # Tipos compartidos
docs/         # Documentación técnica y de diseño
```
